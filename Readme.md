# Streamlit ChatGPT-Style Mock Chat App

This is a simple ChatGPT-style chat application built with Python and Streamlit.

## Features

- Uses `st.chat_message`
- Uses `st.chat_input`
- Stores chat history in `st.session_state.messages`
- Generates mock assistant replies locally
- Does not use any external LLM or third-party API

Install dependencies:
